This page holds comments and links about submission 1406 to IntSys12v5n34. The main links related to the submission are the following:

  • the repository on github that contains all the code and can be cloned freely by anyone, the NOTES file contains some commands to run to experiment,
  • a zip file containing all jars, both dependencies and compiled projects (to avoid the need for building everything), it should be unziped at the root of the repository,
  • a zip with omiscidgui and the necessary plugin preinstalled, also to be unziped in the repository,
  • the Android apk file for the very simple key exporter Android application,
  • the web page of reference [1].
